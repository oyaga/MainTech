# Project Overview

This is a Nuxt.js v3 project for a company called Maintech Solutions. It's a modern web application built with Vue.js and TypeScript, using Tailwind CSS for styling. The project is structured with a component-based architecture, with different sections of the website broken down into individual Vue components.

## Building and Running

### Dependencies

The project uses `npm` to manage dependencies. To install them, run:

```bash
npm install
```

### Development

To start the development server, which will be available at `http://localhost:3000`, run:

```bash
npm run dev
```

### Production

To build the application for production, run:

```bash
npm run build
```

To preview the production build locally, run:

```bash
npm run preview
```

## Development Conventions

*   **Framework:** The project uses Nuxt.js, a framework for Vue.js.
*   **Language:** The primary language is TypeScript.
*   **Styling:** Styling is done using Tailwind CSS.
*   **Component-Based Architecture:** The application is built using a component-based architecture. Components are located in the `app/components` directory and are organized by page or feature.
*   **Pages:** The application's pages are located in the `app/pages` directory.
*   **Static Assets:** Static assets like images and fonts are located in the `public` directory.
